Welcome to Synchro-device documentation!
===================================

**Synchro-device** is a tool for bringing more statistical power to EEG experiments, icreasing control, preventing information loss in brain research due to jittering signals,  and eliminating repetitive frustrating tasks.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
