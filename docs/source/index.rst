Welcome to Synchro-device documentation!
===================================

**Synchro-device** (/lu'make/) is a tool for prevent formation loss in brain research due to jittering signals, bringing more statistical power to EEG experiments, icreases control, and eliminating repetitive frustrating tasks.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
